# planch (Power _Plan Changer_)

<img width="277" height="200" alt="Screenshot of planch (power plan changer)" src="https://github.com/user-attachments/assets/9a78783c-34c2-483c-b1fc-fde6e7936c70" />

Pretty simple program written in C++ and Qt to change the active power plan in Windows.

Right click the icon and then select the power plan you want. 

The icon (should) change depending on if you have a dark or light taskbar.
